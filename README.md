## Project Overview 
This project is a personal web resume built using HTML, CSS, JavaScript, and basic jQuery
It showcases my profile, education, skills, projects, and contact information in a clean, interactive layout. Features include circular profile images, centered content, pop-ups for links, dark/light mode toggle, and an extended contact page.
